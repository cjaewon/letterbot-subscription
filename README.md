### ⬇️ 설치

#### Run
```bash
go run main.go
```

#### Compile html
```bash
npm install -g parcel-bundler
npm run parcel / npm run pacel:watch
```

### 🛠️ 기술 스택
- echo (golang web framework)
- parcel
- typescript
- Mysql, Mariadb
